Feel analyzer

O que é?

O Feel analyzer é um sistema desenvolvido em Python com o framework Flask, que utiliza os serviços da IBM Cloud para fazer uma busca no twitter sobre a palavra digitada, e então, através do serviço Natural Language Understand analiza as emoções relacionadas com os tweets e os sentimentos presentes nos mesmos.

Como utilizar?

Para utilizar, basta digitar o nome ou produto desejado na caixa de texto e clicar no botão para enviar, e então, a análise será feita. OBS: A análise é feita apenas com tweets em inglês, pois o serviço utilizado para analisar emoções ainda não está disponível em português. Então, caso um tweet esteja em outra língua, o sistema irá traduzir, utilizando o Language Translator.
